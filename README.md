- номер рейсу
- пункт призанчення.
- година відправлення
- хвилина відправлення

Заповнити номер потягів. 4 цифри.
Заповнити Пункти пизанчення випадково вибранними містами.
Заповнити години відправлення від 0 до 23 включно.
